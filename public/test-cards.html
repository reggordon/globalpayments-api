3D Secure Testing

  • Use 3DS enrolled cards to test authentication challenges
  • In sandbox, you'll see a test authentication page from Global Payments
  • Any password will work in the test 3DS interface
  • 3DS adds an extra security layer for card-not-present transactions

Testing Tips

  • Amounts: Use any amount - sandbox doesn't process real money
  • Currencies: Test with EUR, USD, GBP, and other supported currencies
  • Multiple Attempts: You can reuse the same card multiple times
  • Transaction History: All test transactions are logged for review
  • Result Codes: Check transaction details to see exact response codes
