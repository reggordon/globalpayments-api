Understanding Result Codes

  • Code 00: Successful transaction - payment approved
  • Code 101: Declined by bank - insufficient funds or other bank restrictions
  • Code 102: Referral B - requires merchant to contact bank
  • Code 103: Card reported lost or stolen - transaction blocked for security
  • Code 111: Strong Customer Authentication (SCA) Required - needs 3DS verification
  • Code 200: Communication Error - connection issue with payment processor
  • Code 205: Communication Error - specific to UATP cards

Testing Tips

  • Amounts: Use any amount - sandbox doesn't process real money
  • Currencies: Test with EUR, USD, GBP, and other supported currencies
  • Multiple Attempts: You can reuse the same card multiple times
  • Transaction History: All test transactions are logged for review
  • Result Codes: Check transaction details to see exact response codes
